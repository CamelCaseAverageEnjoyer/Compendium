###### <u>Структура репозитория</u>
1. MatlabCompendium
    1. **DeployOpticalNav**
    2. **YoloOpticalNav**
2. PythonCompendium
    1. **LogisticGraph1**: для каждого узла заданного графа рассчитывается путь наискорейшего прохождения в любую точку
    2. **LogisticGraph2**: генерит docx-файлики для заданного графа, там примеры инпута в стороже, результат в корне
    3. **MilkFriend**: обработка мороженного
    4. **MyCarEngine**: впустую потрачённое время
    5. **MyVedo**: анимация полёта с нотами реализма
    6. **NearEarthCoordinates**: перевод между системами координат с дальнейшим преобразованием как локальную библиотеку через wheel
3. RnD-master: магистрский НИР
